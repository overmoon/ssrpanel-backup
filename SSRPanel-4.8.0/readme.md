# SSRPanel
Support but not limited to: Shadowsocks,ShadowsocksR,ShadowsocksRR,V2Ray

## Requirements
- PHP 7.1.3 +
- Mysql 5.5 +
- Memory 1G +
- Disk 10G +
- PHP Extensions: ZIP、XML、CURL、GD2、FileInfo、OpenSSL、Mbstring、PDO、Tokenizer、Ctype、JSON、BCMath
- Laravel 5.6
